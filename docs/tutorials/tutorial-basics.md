# 基础概念 - 游戏配置Excel MCP服务器

## 🚀 第1步：基础概念（5分钟）

### 什么是MCP服务器？
MCP（Message Passing Control）服务器是一个让AI能够直接操作Excel配置表的服务。它就像一个"超级Excel"，可以通过自然语言命令来读写、查询和分析游戏数据。

### 游戏配置表的重要性
游戏配置表包含：
- **角色属性**：血量、攻击力、防御力、技能
- **装备数据**：武器、防具、饰品属性
- **技能配置**：技能效果、冷却时间、消耗
- **数值平衡**：怪物强度、掉落概率、经验值
- **关卡设计**：地图配置、敌人分布、任务条件

### MCP服务器架构
```
Excel文件 → MCP服务器 → AI工具调用 → 游戏配置操作
```

### 🎯 核心优势
- **直接访问**：无需打开Excel文件，直接操作数据
- **批量处理**：支持一次性修改大量数据
- **查询分析**：支持SQL查询进行数据分析
- **自动化**：可以编写脚本自动执行重复任务

## 💡 快速开始

### 启动MCP服务器
```bash
# 安装后启动
excel-mcp-server-fastmcp
```

### 基本命令格式
```json
{
  "method": "tools/call",
  "params": {
    "name": "command_name",
    "arguments": {
      "param1": "value1",
      "param2": "value2"
    }
  }
}
```
