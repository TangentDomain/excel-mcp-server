## 🚀 SQL引擎能力总结

### ✅ 完全支持的高级特性

1. **窗口函数完整支持**
   - 基础: ROW_NUMBER, RANK, DENSE_RANK, NTILE
   - 偏移: LEAD, LAG
   - 首尾: FIRST_VALUE, LAST_VALUE
   - 范围: ROWS BETWEEN

2. **复杂查询**
   - 多层嵌套子查询
   - CTE (WITH)
   - EXISTS/IN 子查询
   - 自连接
   - UNION/INTERSECT/EXCEPT

3. **表达式组合**
   - 数学函数组合
   - CASE WHEN嵌套
   - 聚合+CASE WHEN

1. **聚合的聚合** - `AVG(AVG(x))`

3. **正则表达式** - `RLIKE`
