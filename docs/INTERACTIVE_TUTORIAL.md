#### 🎯 高级查询功能
- **精确查询**：`query "SELECT * FROM skills WHERE class = '法师'"`
- **关联查询**：跨表JOIN查询职业和技能关联数据
- **聚合分析**：计算各职业技能数量、平均攻击力等

#### 🎮 游戏查询实战

**查询1：技能平衡分析**
```sql
SELECT class, COUNT(*) as skill_count, AVG(attack) as avg_attack
FROM skills 
GROUP BY class
```

**查询2：装备套装查询**
```sql
SELECT e.*, s.set_name, s.bonus
FROM equipment e
LEFT JOIN equipment_sets s ON e.set_id = s.id
WHERE e.rarity = '史诗'
```

**查询3：子查询优化**
```sql
SELECT * FROM skills 
WHERE attack > (SELECT AVG(attack) FROM skills)
```
