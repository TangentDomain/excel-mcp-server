## 📝 已知限制

### SQL标准限制（非bug）
1. WHERE子句无法引用窗口函数别名（SQL执行顺序限制）
2. WHERE column = NULL 返回空结果（SQL标准行为）

### 未实现的高级特性
1. **Tuple比较** - `WHERE (a, b) IN ((1, 2), (3, 4))`
2. **CAST函数** - `CAST(col AS type)`
3. **递归CTE** - `WITH RECURSIVE`

### 错误处理（正确行为）
1. ORDER BY不存在的列 → 报错
2. LIMIT负数 → 报错
3. 除以零 → 返回NULL（需用NULLIF保护）
