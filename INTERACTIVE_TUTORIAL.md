## 🚀 快速开始

### 第1步：安装配置
```bash
# 一键安装
pip install excel-mcp-server-fastmcp

# 验证安装
excel-mcp-server-fastmcp --version
```

**✅ 完成标志**：看到版本号输出 `v1.6.49`

### 第2步：首次启动
```bash
# 启动MCP服务器
excel-mcp-server-fastmcp

# 预期输出：
# 🎮 ExcelMCP 游戏开发专用Excel管理器 v1.6.49
# 📊 53个工具已加载，准备就绪
# 💡 输入 "help" 查看可用工具
```

**✅ 完成标志**：看到启动成功消息

### 技能3：跨表操作 (8分钟)

#### 目标
学会使用JOIN查询关联多个表

#### 场景设置
假设你有以下两个表格：
- **技能表**：技能ID、技能名称、技能类型、消耗MP、伤害值
- **职业表**：职业ID、职业名称、主技能类型、难度等级

#### 练习1：关联查询
**任务**：查询每个职业及其对应的主技能
**输入指令**：
```text
查询职业表和技能表，找出每个职业的主技能，按职业难度排序
```

**预期结果**：
- 显示职业名称 + 对应的主技能
- 按难度等级排序

#### 练习2：复杂分析
**任务**：分析不同职业的技能消耗情况
**输入指令**：
```text
分析职业表和技能表，计算每个职业的平均技能消耗，找出消耗最高的职业
```

**预期结果**：
- 显示每个职业的平均消耗
- 标出消耗最高的职业

#### ✅ 验证清单
- [ ] 能成功执行JOIN查询
- [ ] 能理解关联数据的含义
- [ ] 能进行跨表数据分析

## 🎮 游戏场景实战

### 场景1：技能系统优化 (10分钟)

#### 背景设定
你正在开发一款RPG游戏，需要平衡技能系统：

- 技能表包含：攻击技能、防御技能、治疗技能
- 当前问题：攻击技能过强，需要调整

#### 任务清单
1. **现状分析**
   ```text
   分析技能表，统计每种类型技能的数量和平均伤害
   ```

2. **平衡调整**
   ```text
   将所有攻击技能的伤害降低15%，将所有防御技能的持续时间增加20%
   ```

3. **效果验证**
   ```text
   再次分析技能表，确认调整后的平衡性
   ```

#### ✅ 完成标准
- [ ] 能分析技能分布情况
- [ ] 能成功调整技能平衡
- [ ] 能验证调整效果

## 💡 学习提示

### 🎯 有效练习方法
1. **从简单开始**：先掌握基础查询，再尝试复杂操作
2. **实际场景**：用真实游戏数据练习，效果更好
3. **逐步挑战**：完成基础练习后再尝试进阶挑战
4. **定期复习**：温故知新，巩固技能

### 🔍 常见问题解决
- **查询无结果**：检查表名和字段名是否正确
- **更新失败**：确认数据类型和格式是否正确
- **性能问题**：大文件使用分页查询，避免全量加载

### 📞 获取帮助
- **查看帮助**：输入 "help" 查看所有可用工具
- **查看日志**：检查 `.excel_mcp_logs/` 目录下的详细日志
- **提交问题**：[GitHub Issues](https://github.com/TangentDomain/excel-mcp-server/issues/new)
