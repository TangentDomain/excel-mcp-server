## 🚀 一分钟上手（完全零经验）

### 👶 第零步：我是谁？为什么需要这个？

**如果你是：**
- 🎮 游戏策划 - 需要修改技能/装备/怪物数值
- 🔧 程序员 - 需要管理Excel配置表  
- 📊 数据分析师 - 需要处理游戏数据
- 👨‍💼 项目经理 - 需要查看配置变更

**ExcelMCP能帮你：**
- 🗣️ **用中文说话** - "把所有法师技能攻击力加10%"
- 🔍 **智能查询** - "找出价格>1000的史诗装备"
- ⚡ **批量操作** - 一次修改整张表的多个数据
- 🎯 **跨表关联** - 连接技能表和职业表进行统计

### 💫 第一步：验证安装（30秒）

```bash
# 检查是否已安装
excel-mcp-server-fastmcp --version

# 如果没安装（20秒搞定）
pip install excel-mcp-server-fastmcp
```

**✅ 成功标志**：显示 `v1.6.51`

### 🎮 第二步：首次对话（30秒）

在AI客户端输入：
```bash
"读取技能表，显示前3个技能的名称和攻击力"
```

**🎯 期望结果**：
- 正确识别你的Excel文件
- 返回技能数据表格
- 如果出错，会告诉你具体问题

## 🎮 三分钟完整体验

### 📋 准备工作（1分钟）
1. **准备Excel文件**
   ```bash
   # 下载测试文件（如果暂时没有自己的）
   wget https://example.com/game_data.xlsx
   ```

2. **启动服务**
   ```bash
   excel-mcp-server-fastmcp
   # 看到 "53个工具已加载" 表示成功
   ```

### 🗣️ 试试这些命令（2分钟）

#### 基础查询（不会错）
```bash
# 1. 查看所有sheet
"显示我的Excel里有哪些工作表"

# 2. 查看数据结构  
"查看技能表的前5行数据"

# 3. 查看列名
"技能表有哪些列？"
```

#### 进阶查询（展示威力）
```bash
# 4. 条件查询
"找出攻击力>100的法师技能"

# 5. 统计分析
"统计每个职业的技能数量"

# 6. 跨表关联
"连接技能表和职业表，显示职业名称和技能数量"
```

### 🎯 常见问题快速解决（1分钟）

#### ❌ "找不到文件"
**问题**：AI找不到你的Excel文件
**解决**：
```bash
# 告诉AI文件路径
"读取 /path/to/your/skills.xlsx"
```

#### ❌ "看不懂你的表格"
**问题**：AI不认识表格结构
**解决**：
```bash
# 先描述表结构
"技能表有：技能ID、技能名称、职业、攻击力、冷却时间"
```

#### ❌ "数据格式错误"
**问题**：某些列数据格式有问题
**解决**：
```bash
# 让AI修复格式
"修复技能表的攻击力列，确保都是数字"
```

## 💡 新手必知技巧

### 🎯 最常用的3个命令
```bash
# 1. 查看数据
"显示XX表的前5行"

# 2. 条件查询  
"找出XX>100的数据"

# 3. 批量修改
"把所有XX的值增加10"
```

### 🚫 避免的3个坑
1. **不要用英文列名** - 用"技能ID"而不是"skill_id"
2. **不要太复杂** - 先简单查询，再复杂操作
3. **先备份** - 重要数据先复制一份再修改

### 🎉 成功标志
当你能熟练说出：
- "把法师技能攻击力加20%"
- "找出价格最高的装备"
- "统计每个职业的技能数量"

恭喜！你已经掌握了ExcelMCP的核心技能！
